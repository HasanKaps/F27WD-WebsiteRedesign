Best Atari / Best Atari Light Gun Tip.

 

All Atari Light Gun 8 bit computer Games and Atari 2600 / 7800 Light gun games were designed to work only with a Normal Vacuum Tube type of TV's or Monitors displays!  They will not work with any of the newer LCD / Plasma / TV's and Projection Type of TV / Monitors.

 

Only recently on some of the State of the Art (XBOX 360, PlayStation 3 and so on) Current non Atari Game systems have new Light Guns and Light Gun Games that have been developed to work on the Newer LCD / Plasma TV's
